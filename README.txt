
Минималистическая версия Vik-Off-CMF, содержащая лишь набор базовых классов:
	- FrontController для обработки запросов и вызова соответствующих методов;
	- Db (вместе с адаптерами) для работы с базой данных;
	- Layout для удобной работы с отображением.

Для создания основы нового проекта, используйте install.php (запуск из командной строки).
Запущенный без аргументов сценарий подскажет что надо делать.
